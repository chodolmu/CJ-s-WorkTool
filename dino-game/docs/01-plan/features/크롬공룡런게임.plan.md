# Plan: 크롬공룡런게임

## Executive Summary

| 항목 | 내용 |
|------|------|
| Feature | 크롬 공룡 런게임 (Chrome Dino Run Game) |
| 시작일 | 2026-03-31 |
| 목표 | 싱글 HTML 파일로 크롬 오프라인 공룡 게임 완전 재현 |

### Value Delivered (4-Perspective)

| 관점 | 내용 |
|------|------|
| Problem | 크롬 오프라인 게임을 별도 설치 없이 어디서든 즐기고 싶다 |
| Solution | 바닐라 JS + Canvas API로 완전 재현한 싱글 HTML 파일 게임 |

## 1. 기능 정의

### 1.1 핵심 기능 (MVP)

| 우선순위 | 기능 | 설명 |
|---------|------|------|
| P0 | 게임 캔버스 초기화 | HTML5 Canvas 세팅, 게임 루프 구성 |
| P0 | 공룡 캐릭터 렌더링 및 물리 | 중력, 점프, 착지 구현 |
| P0 | 지면 및 배경 스크롤 | 무한 스크롤 지면 렌더링 |
| P0 | 입력 처리 (점프) | Space/ArrowUp/터치로 점프 |
| P0 | 선인장 장애물 생성 및 이동 | 랜덤 간격 선인장 생성 및 좌측 이동 |
| P0 | AABB 충돌 감지 | 공룡-장애물 충돌 판정 |
| P0 | 게임 상태 관리 | 대기/실행/게임오버 상태 전환 |
| P0 | 점수 시스템 | 생존 시간 기반 점수 증가 |
| P1 | 게임 오버 화면 및 재시작 | 게임오버 메시지 + 재시작 입력 |
| P1 | 하이스코어 저장 | localStorage 기반 최고점수 저장 |
| P1 | 속도 점진적 증가 | 점수에 따른 게임 속도 증가 |
| P1 | 숙이기(덕) 액션 | ArrowDown으로 숙이기 |
| P2 | 새(익룡) 장애물 | 공중 장애물 추가 |
| P2 | 사운드 효과 | Web Audio API 점프/게임오버 효과음 |
| P2 | 구름 및 장식 요소 | 배경 구름 스크롤 |
| P2 | 다크/라이트 모드 전환 | 야간 모드 자동 전환 |
| P2 | 별(달) 야간 장식 | 야간 모드 별/달 장식 |

### 1.2 기술 스택

- **렌더링**: HTML5 Canvas API
- **언어**: 바닐라 JavaScript (ES6+)
- **스타일**: CSS3 (인라인)
- **배포**: 싱글 HTML 파일 (index.html)
- **저장소**: localStorage (하이스코어)
- **오디오**: Web Audio API (효과음)

### 1.3 게임 스펙

| 항목 | 값 |
|------|----|
| 캔버스 크기 | 800 x 300px |
| 초기 속도 | 6 px/frame |
| 최대 속도 | 13 px/frame |
| 중력 | 0.6 |
| 점프력 | -12 |
| 프레임레이트 | 60 FPS |
| 점수 단위 | 100ms마다 +1 |

## 2. 구현 순서

### Phase 1: 코어 엔진
1. HTML 구조 + Canvas 초기화
2. 게임 루프 (requestAnimationFrame)
3. 공룡 클래스 (렌더링 + 물리)
4. 지면 스크롤

### Phase 2: 게임플레이
5. 입력 처리 (키보드 + 터치)
6. 선인장 장애물 시스템
7. 충돌 감지
8. 점수 + 게임 상태 관리

### Phase 3: 폴리시
9. 게임오버 화면 + 재시작
10. 하이스코어 + localStorage
11. 속도 증가 곡선
12. 숙이기 액션

### Phase 4: 고급 기능
13. 익룡 장애물
14. 구름 + 장식 요소
15. 다크/야간 모드
16. 사운드 효과
